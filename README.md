# PokéTracker

A web application for tracking and managing your Pokémon collection.

## Live Demo

🔗 [Visit PokéTracker](https://pokedex-tracker.vercel.app/)

## About

PokéTracker helps you organize and track your Pokémon collection with an intuitive interface.

## Tech Stack

- **Frontend**: React, JavaScript
- **Backend**: PostgreSQL with Supabase
- **Authentication**: Supabase Auth
- **API**: RESTful API
